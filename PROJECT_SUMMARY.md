## 📝 Customization

### 1. Update Company Info
Edit: `src/config/site-config.ts`
- Change company name
- Update hero content
- Add your services
- Update team members
- Add testimonials

### 2. Modify Colors
Edit: `tailwind.config.ts`
- Change primary accent color (currently blue)
- Adjust neutral palette
- Update background colors

### 3. Add More Sections
Copy any existing section and customize:
- Follow the existing pattern
- Use same animation config
- Keep styling consistent

### 4. Deploy
```bash
npm run build
# Upload to: Vercel, Netlify, or any Node.js host
```
